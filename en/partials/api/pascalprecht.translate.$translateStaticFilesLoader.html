$translateStaticFilesLoader
service in module pascalprecht.translate

Description

Creates a loading function for a typical static file url pattern: "lang-en_US.json", "lang-de_DE.json", etc. Using this builder, the response of these urls must be an object of key-value pairs.

Dependencies

  • $q
  • $http

Usage

$translateStaticFilesLoader(options);

Parameters

ParamTypeDetails
optionsobject

Options object, which gets prefix, suffix and key.
