i18n для ваших Angular приложений - просто

####Простейший способ добавить i18n и l10n в ваши приложения!


Вы хотите добавить i18n и l10n к вашим приложениям и не можете найти простой способ, как это сделать? Вам поможет angular-translate. Он поставляется с очень полезными функциями и расширениями, а также проявляет ультра гибкость, когда дело доходит до настройки. Вот лишь некоторые из его замечательных возможностей:

Возможности

  • Компоненты для перевода контента (фильтры/директивы)
  • Асинхронная загрузка i18n данных
  • Поддержка плюрализации с помощью MessageFormat.js
  • Расширяемость через простые в использовании интерфейсы

Если вы полюбили этот модуль и используете его в своих проектах, пожалуйста, обнародуйте его на ngmodules и кликните на кнопку "I use it"! angular-translate легко установить и еще проще использовать! Прочитав это руководство вы научитесь использовать angular-translate и его замечательные функции! Давайте же начнем!

Быстрый старт

Установите модуль с помощью Bower:

$ bower install angular-translate

Убедитесь, что подключили его в HTML документ:

<script src="path/to/angular-translate.js"></script>

Внедрите модуль angular-translate как зависимость в ваше приложение:

var app = angular.module('myApp', ['pascalprecht.translate']);

Научите ваше приложение переводам:

app.config(['$translateProvider', function ($translateProvider) {
  $translateProvider.translations('en', {
    'TITLE': 'Hello',
    'FOO': 'This is a paragraph',
  });

  $translateProvider.translations('de', {
    'TITLE': 'Hallo',
    'FOO': 'Dies ist ein Absatz'
  });

  $translateProvider.preferredLanguage('en');
}]);

Переведите ваше приложение:

<h1>{{ 'TITLE' | translate }}</h1>
<p>{{ 'FOO' | translate }}</p>

Это просто быстрый пример того, как работает angular-translate. В целом, все это работает так: "предоставить данные i18n, дать о них знать angular-translate и использовать его компоненты для интерполяции ваших переводов некоторыми значениями.". Просто, да?
