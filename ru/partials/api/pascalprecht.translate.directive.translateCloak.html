translateCloak
directive in module pascalprecht.translate

Dependencies

  • $rootScope
  • $translate

Usage

as attribute
<ANY translate-cloak
     [translate-cloak="{string}"]>
   ...
</ANY>
as class
<ANY class="translate-cloak [translate-cloak: {string};]">
   ...
</ANY>

Parameters

ParamTypeDetails
translate-cloak
(optional)
string

If a translationId is provided, it will be used for showing or hiding the cloak. Basically it relies on the translation resolve.
