國際化你的 Angular 應用程式是如此的輕鬆

最簡單的方法國際化和當地語系化你的 Angular 應用程式！


你想找到一個國際化和當地語系化你的angular應用程式，卻找不到一個簡單的方法麼？ angular-translate可以提供你所需的服務。它配備了豐富實用的功能和擴展，並為您提供超高的靈活性和可定制性。 這些都是偉大的功能，一下只是簡單列舉幾個：

功能

  • 通過元件（篩檢程式/指令）來翻譯你的內容
  • 非同步載入國際化的資料
  • 採用多元化的支援 MessageFormat.js
  • 通過便於使用的介面提供可擴充性

如果你喜歡的這個模組，並把它用在你的專案中，請將這個資訊公開在 ngmodules，並按一下 "I use it" 按鈕！angular-translate 易於安裝， 使用起來更加得心應手！ 當您閱讀本文時，您將瞭解如何使用 angular-translate 和他擁有的很酷的功能！ 讓我們開始吧：

快速入門

通過 bower 安裝模組：

$ bower install angular-translate

確保將它嵌入在你的HTML文檔中。

<script src="path/to/angular-translate.js"></script>

將 angular-translate 模組作為依賴注入到你的應用程式：

var app = angular.module('myApp', ['pascalprecht.translate']);

教你的應用程式去翻譯：

app.config(['$translateProvider', function ($translateProvider) {
$translateProvider.translations('en', {
'TITLE': 'Hello',
'FOO': 'This is a paragraph'
});

$translateProvider.translations('de', {
'TITLE': 'Hallo',
'FOO': 'Dies ist ein Absatz'
});

$translateProvider.preferredLanguage('en');
}]);

翻譯你的應用程式：

<h1>{{ 'TITLE' | translate }}</h1>
<p>{{ 'FOO' | translate }}</p>

這是一個 angular-translate 如何翻譯的簡單的例子。基本上整個事情就像「提供一些國際化的資料，讓 angular-translate 知道它 並使用其元件來（對某些值）插值入你的翻譯。“。容易，不是麼？
