$translate
service in module pascalprecht.translate

Description

The $translate service is the actual core of angular-translate. It expects a translation id and optional interpolate parameters to translate contents.

 $translate('HEADLINE_TEXT').then(function (translation) {
   $scope.translatedText = translation;
 });

Dependencies

  • $log

  • refresh(langKey)

    Refreshes a translation table pointed by the given langKey. If langKey is not specified, the module will drop all existent translation tables and load new version of those which are currently in use.

    Refresh means that the module will drop target translation table and try to load it again.

    In case there are no loaders registered the refresh() method will throw an Error.

    If the module is able to refresh translation tables refresh() method will broadcast $translateRefreshStart and $translateRefreshEnd events.

    Parameters
    ParamTypeDetails
    langKeystring

    A language key of the table, which has to be refreshed

    Returns
    promise

    Promise, which will be resolved in case a translation tables refreshing process is finished successfully, and reject if not.

    Example

    // this will drop all currently existent translation tables and reload those which are // currently in use $translate.refresh(); // this will refresh a translation table for the en_US language $translate.refresh('en_US');

  • use([key])

    Tells angular-translate which language to use by given language key. This method is used to change language at runtime. It also takes care of storing the language key in a configured store to let your app remember the choosed language.

    When trying to 'use' a language which isn't available it tries to load it asynchronously with registered loaders.

    Returns promise object with loaded language file data or string of the currently used language.

    If no or a falsy key is given it returns the currently used language key. The returned string will be undefined if setting up $translate hasn't finished.

    Parameters
    ParamTypeDetails
    [key]string

    Language key

    Returns
    object|string

    Promise with loaded language data or the language key if a falsy param was given.

    Example

    $translate.use("en_US").then(function(data){ $scope.text = $translate("HELLO"); });
