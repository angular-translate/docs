国际化你的 Angular 应用程序是如此的轻松

最简单的方法国际化和本地化你的 Angular 应用程序！


你想找到一个国际化和本地化你的angular应用程序，却找不到一个简单的方法么？ angular-translate可以提供你所需的服务。它配备了丰富实用的功能和扩展，并为您提供超高的灵活性和可定制性。 这些都是伟大的功能，一下只是简单列举几个：

功能

  • 通过组件（过滤器/指令）来翻译你的内容
  • 异步加载国际化的数据
  • 采用多元化的支持 MessageFormat.js
  • 通过易于使用的接口提供可扩展性

如果你喜欢的这个模块，并把它用在你的项目中，请将这个信息公开在 ngmodules，并单击 "I use it" 按钮！angular-translate 易于安装， 使用起来更加得心应手！ 当您阅读本文时，您将了解如何使用 angular-translate 和他拥有的很酷的功能！ 让我们开始吧：

快速入门

通过 bower 安装模块：

$ bower install angular-translate

确保将它嵌入在你的HTML文档中。

<script src="path/to/angular-translate.js"></script>

将 angular-translate 模块作为依赖注入到你的应用程序：

var app = angular.module('myApp', ['pascalprecht.translate']);

教你的应用程序去翻译：

app.config(['$translateProvider', function ($translateProvider) {
  $translateProvider.translations('en', {
    'TITLE': 'Hello',
    'FOO': 'This is a paragraph'
  });

  $translateProvider.translations('de', {
    'TITLE': 'Hallo',
    'FOO': 'Dies ist ein Absatz'
  });

  $translateProvider.preferredLanguage('en');
}]);

翻译你的应用程序：

<h1>{{ 'TITLE' | translate }}</h1>
<p>{{ 'FOO' | translate }}</p>

这是一个 angular-translate 如何翻译的简单的例子。基本上整个事情就像“提供一些国际化的数据，让 angular-translate 知道它 并使用其组件来（对某些值）插值入你的翻译。“。容易，不是么？
